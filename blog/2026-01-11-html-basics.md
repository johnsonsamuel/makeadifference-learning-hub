## Your First Step: HTML Basics

HTML defines the structure of a webpage.

### Example

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello Frontend!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>
```
